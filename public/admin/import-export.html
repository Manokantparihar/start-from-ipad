📖 CSV Format Guide (click to expand)

Required columns: quizTitle, mode, question, option1, option2, correctAnswer

Optional columns: topic, difficulty, timeLimit, isPublished, option3, option4, explanation, marks, negativeMarks, adminNotes

Rules:

  • One row = one question. Rows with the same quizTitle belong to the same quiz.
  • mode must be: daily, topic, or mock
  • difficulty must be: easy, medium, or hard (default: medium)
  • correctAnswer must exactly match one of option1–option4
  • isPublished: true or false (default: false)
  • Maximum file size: 5 MB
  • Quizzes with a title that already exists will be skipped (no overwrite)

Example row:

quizTitle,mode,topic,difficulty,timeLimit,isPublished,question,option1,option2,option3,option4,correctAnswer,explanation,marks
