# 🗺️ Programming Learning Roadmap — 90 Day Plan (Python-first)

Track progress while learning programming from scratch with the GitHub Education Pack, prioritizing Python as the first language. Check items as you complete them.

---

## 🎯 Month 1 — Foundations (Days 1–30)

### Week 1–2: Setup & Git Basics
- [ ] Set up VS Code + GitHub Copilot extension
- [ ] Complete GitHub Skills: Introduction to GitHub + Git Basics
- [ ] Install Python 3.12+, pip, and create a `venv`
- [ ] Practice Git essentials: clone, commit, push, pull, branch
- [ ] Run first Python script (`print("hello")`) locally and in Codespaces

### Week 3–4: Python Core
- [ ] Learn Python syntax: variables, control flow, functions, modules
- [ ] Master data structures: lists, dicts, sets, tuples, comprehensions
- [ ] Finish a beginner course (freeCodeCamp Python or CS50P)
- [ ] Build mini-scripts: unit converter, text formatter, simple calculator
- [ ] Start a daily learning log in the repo (short notes in README or issues)

---

## 🚀 Month 2 — Build Projects (Days 31–60)

### Week 5–6: Data & APIs
- [ ] Learn file I/O, JSON, and error handling in Python
- [ ] Practice `requests` + `asyncio` basics for API fetching
- [ ] Build Project #1: CLI Weather/Notes app (reads/writes files, fetches an API)
- [ ] Write a clear README and push to GitHub with a tagged release
- [ ] Add a simple test script (doctest or `pytest -q`) for core functions

### Week 7–8: Web Basics & Automation
- [ ] Learn a lightweight web framework (Flask or FastAPI) and routing
- [ ] Build Project #2: To-Do or Habit Tracker API (CRUD + JSON storage/SQLite)
- [ ] Add a tiny HTML/CSS front or a CLI client to consume your API
- [ ] Set up GitHub Actions for lint/test on push (use Codespaces template)
- [ ] Claim a free domain from the Education Pack and map it to a simple static page

---

## 🏆 Month 3 — Level Up (Days 61–90)

### Week 9–10: Full Stack & Database
- [ ] Deepen Python: virtualenv hygiene, packaging, logging, and typing
- [ ] Add SQLite + SQLAlchemy/SQLModel to your API for persistence
- [ ] Build Project #3: Full-stack app (FastAPI/Flask backend + minimal frontend)
- [ ] Explore auth basics (sessions or tokens) and secure form handling
- [ ] Write deployment scripts (Procfile/Dockerfile or `uvicorn` command)

### Week 11–12: Portfolio & Deploy
- [ ] Deploy a project to Render/Azure/DigitalOcean credits from the pack
- [ ] Connect custom domain to your deployed app and verify HTTPS
- [ ] Polish portfolio site with your best 6 repos pinned on GitHub
- [ ] Add detailed READMEs with setup, run, and test steps for each project
- [ ] Explore Python next-steps: `pandas` + `matplotlib` or `scikit-learn` basics

---

## 📚 GitHub Education Pack Resources (Python-friendly)
- [ ] GitHub Copilot — AI pair programmer for Python
- [ ] GitHub Codespaces — prebuilt dev environment with VS Code + terminal
- [ ] freeCodeCamp Python + APIs curriculum — fundamentals and projects
- [ ] CS50P (Python) — computer science foundations
- [ ] Educative.io — interactive Python/algorithms tracks
- [ ] Frontend Masters + The Odin Project — HTML/CSS/JS when ready for frontend

---

## 📈 Daily Habits
- [ ] Code in Python for at least 30 minutes daily
- [ ] Make at least 1 GitHub commit daily (keep the streak going! 🟩)
- [ ] Read/watch 1 tutorial daily; summarize in a short log
- [ ] Note what you practiced in commit messages (problem + takeaway)
